# Advanced Pandas Course Outline

This document describes the proposed lesson structure for a beginner-to-advanced pandas course.

## Course File Structure

- 01_introduction_to_pandas.md
  - What pandas is, why it matters, install/setup, core data structures, and the mindset of data analysis.
- 02_series_and_indexing.md
  - Series creation, indexing, slicing, labels, alignment, and basic operations.
- 03_dataframes_and_basic_operations.md
  - DataFrame creation, columns, rows, selecting data, filtering, sorting, and basic inspection.
- 04_loading_and_exporting_data.md
  - Reading CSV/Excel/JSON/SQL data, writing files, handling paths, and working with real datasets.
- 05_data_types_missing_values_and_cleaning.md
  - dtypes, object conversion, missing values, duplicates, string cleaning, and type normalization.
- 06_selection_filtering_and_grouping_basics.md
  - Boolean masks, loc/iloc, conditional filtering, and foundational grouping concepts.
- 07_aggregation_groupby_and_pivot_tables.md
  - groupby, split-apply-combine, aggregation, pivot tables, and summary statistics.
- 08_merge_join_and_concatenation.md
  - Combining datasets using merge, join, concat, append, and relational operations.
- 09_datetime_and_time_series_analysis.md
  - Datetime parsing, time-based indexing, resampling, rolling windows, and trend analysis.
- 10_text_string_operations_and_categorical_data.md
  - String methods, regex, category dtype, and text-based cleaning workflows.
- 11_reshaping_and_multi_indexing.md
  - melt, stack/unstack, wide-to-long transformations, hierarchical indexes, and reshape strategies.
- 12_window_functions_and_advanced_analytics.md
  - rolling, expanding, rank, cumulative operations, and advanced analytical patterns.
- 13_advanced_missing_data_and_outlier_handling.md
  - imputation, interpolation, anomaly detection, outlier treatment, and robust data quality workflows.
- 14_performance_optimization_and_vectorization.md
  - Vectorized operations, avoiding loops, memory use, dtype optimization, and performance profiling.
- 15_advanced_data_engineering_with_pandas.md
  - ETL-style workflows, pipeline design, scalable transformations, and production-oriented operations.
- 16_exploratory_data_analysis_and_visualization.md
  - EDA workflows, descriptive analysis, correlations, distributions, and integrating pandas with plotting libraries.
- 17_scalability_big_data_workflows_and_best_practices.md
  - chunking, memory management, working with large datasets, and pandas best practices.
- 18_capstone_project_and_final_review.md
  - A full project that combines cleaning, transformation, analysis, and reporting in a realistic business case.

## Planned Learning Progression

The course starts with the basics of pandas objects and moves toward advanced data engineering, performance optimization, and real-world project workflows.
